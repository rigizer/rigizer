# 한재용 (Jaeyong Han, 韓在鎔)
```
안녕하세요! Frontend, Backend, DevOps, AI에 관심있는 개발자입니다.

[Frontend]
- 안드로이드 앱 개발 시 Java, Kotlin, Flutter(Dart) 사용이 가능합니다.
- 학교 학식 앱을 직접 개발, PlayStore 배포하여 1,200명이 넘는 다운로드 횟수를 기록했습니다.
- BootPay 모듈을 연결하여 카드결제 및 카카오페이 결제할 수 있도록 개발한 경험이 있습니다.
- BootStrap, Tailwind, Chart.js를 활용하여 Web을 구성합니다.
- PC와 모바일 웹을 모두 대응하는 반응형 웹과, 화이트 모드와 다크 모드 모두에 대응하는 App.을 개발합니다.
- UI가 단순히 예쁜 것보다, UX가 더 편한 것을 추구합니다.

[Backend]
- Java Spring(JSP, Boot), Python (Flask, FastAPI), Node.js를 이용하여 API 개발이 가능합니다.
- MVC 패턴 기반의 프로젝트 구조를 형성하며, 공통 함수는 유틸리티로 별도 관리합니다.
- Spring Security를 이용하여 JWT Token 기반의 Access Token과 Refresh Token을 관리하여 보안을 향상시킵니다.
- 가상환경 및 환경변수를 활용하여 로컬 개발환경과 배포 환경을 구분합니다.
- SSAFY 5개 캠퍼스에 해당하는 점심 메뉴를 1,000명이 넘는 사용자에게 알림을 보낸 경험이 있습니다.
- AWS S3을 이용하여 첨부파일을 관리하고, AWS DynamoDB와 RDS (MySQL)을 이용하여 데이터를 관리했습니다.
- 웹 스크래핑 서비스를 현업에서 개발한 경험이 있으며, 일 100만 건 이상의 데이터를 처리했습니다.

[DevOps]
- 서비스를 Dockerize하거나, docker-compose.yml을 이용하여 다중 서비스를 배포할 수 있게 스크립트를 작성합니다.
- Docker 내부 네트워크를 활용하고, Nginx Proxy Manager를 이용해 SSL과 도메인을 관리합니다.
- 직접 운영하고 있는 RaspberryPi와 Oracle Cloud 기반의 사나래(sanarae.dev) 서버에 서비스를 배포합니다.
- Github Actions, Gitlab CI/CD, Jenkins를 이용해서 자동 배포를 구현합니다.
- 필요한 경우, Nginx의 리버스 프록시를 이용해서 블루/그린 기반의 무중단 서비스를 구현할 수 있습니다.
- 리눅스(Debian, RedHat) 기반의 명령어를 숙지하고 있으며, 노트북에 리눅스를 설치하여 일상적으로 명령어를 활용하고 있습니다.
- AWS의 EC2와 Lightsail을 이용해 IaaS 기반 배포가 가능하며, 필요에 따라 AWS Lambda와 같은 서버리스 배포도 가능합니다.

[AI]
- 직접 구축한 사나래(sanarae.dev) 서버에 로컬 LLM(gemma3:1b)를 구성하여 개인 AI 비서를 활용합니다.
- AI Hub에서 필요한 데이터를 다운로드 받아, PyTorch/Tensorflow를 이용하여 데이터를 학습하거나, 결과를 예측할 수 있습니다.
- n8n을 활용하여 AI Workflow를 구성합니다.

[Etc.]
- 정규표현식을 이용하여 이름, 전화번호, 아이디, 비밀번호에 대한 규칙을 작성할 수 있습니다.
- Jira를 이용해서 프로젝트 일정을 관리하며, Git 커밋 시 Slack에 알림이 가게 하여 코드 리뷰를 바로바로 진행합니다.
- 통화 서비스를 구현하기 위해 직접 aar 드라이버를 빌드한 경험이 있습니다. (통화 중 TTS, STT 구현)
- 개발 관련 문서를 꾸준히 작성하여, 한 번 이상 경험한 오류에 대해 능숙하게 대처합니다.
- 주기적인 알고리즘 및 CS 스터디를 진행하여 반복 학습을 진행 중입니다.
- SSAFY 7기부터 15기까지 모여있는, 약 1,200명의 교육생과 함께 알고리즘 톡방을 운영하고 있습니다.
- 위에 적어둔 모든 것들은 LLM AI의 도움 없이도 개발이 가능하며, 필요한 경우 Claude Code, Codex, Gemini-CLI 등의 도움을 받아 더 빠르게 개발이 가능합니다.
```
